# Escrevendo as Classes de um Jogo

Este projeto foi desenvolvido como parte de um desafio de lógica de programação da DIO. O objetivo é aplicar conceitos fundamentais de JavaScript para criar uma classe capaz de representar diferentes tipos de heróis de um jogo de aventura.

## Objetivo

O projeto possui uma classe genérica chamada `Heroi`, responsável por representar um personagem com as seguintes propriedades:

- Nome
- Idade
- Tipo

Cada herói também possui um método chamado `atacar()`. Esse método identifica o tipo do personagem e exibe uma mensagem informando qual ataque foi utilizado.

## Tipos de heróis e ataques

| Tipo | Ataque |
|---|---|
| Mago | Magia |
| Guerreiro | Espada |
| Monge | Artes marciais |
| Ninja | Shuriken |

## Como executar

Clone o repositório:

```bash
git clone https://github.com/cmosantos/desafio-classes-de-um-jogo.git
```

Acesse a pasta do projeto:

```bash
cd desafio-classes-de-um-jogo
```

Execute o arquivo com Node.js:

```bash
node src/index.js
```
